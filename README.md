# 🥛 Fresh Milk Delivery & Billing System

A smart Android application designed to digitalize and automate the traditional milk delivery and billing process for vendors and customers.

---

## 📱 Project Overview

This application replaces manual record-keeping with a digital system that helps vendors manage daily milk delivery, billing, payments, and customer data efficiently.

---

## 🚀 Current Features (Implemented)

### 👨‍🌾 Vendor Panel
- Dashboard (Total Customers, Daily Delivery, Revenue)
- Customer Management (Add / Edit / Delete)
- Daily Milk Entry (Morning / Evening)
- Monthly Bill Generation
- PDF Bill Download
- Extra Charges Feature (Ghee, Pending Due, etc.)
- Payment Verification (Screenshot-based)
- Reports & Charts (Monthly Earnings, Customer Data)
- Payment Reminder System

---

### 👤 Customer Panel
- View Daily Milk Entries
- View Monthly Bill
- Download PDF Bill
- Payment Options:
  - UPI
  - QR Code
  - Razorpay (Card/Online)
- Upload Payment Screenshot
- Payment Status System:
  - Pending
  - Processing
  - Paid

## 🛠️ Technologies Used

- Frontend: Android (Java)
- Backend: Firebase Firestore
- Authentication: Firebase Auth
- Payments: Razorpay / UPI / QR Code
- Storage: Cloudinary (for screenshots)
- Charts: MPAndroidChart
